Contiene las imágenes del sitio web, tiene varias subcarpetas por defecto:
    - banner: Imágenes de encabezado o portada
    - galería: Imágenes dinámicas de eventos o actividades (Las que se suban desde
      el panel de administrador)
    - iconos: Iconos de estilos (Menú, redes, etc) - Si se usarán iconos de fontawezome
      se puede omitir
